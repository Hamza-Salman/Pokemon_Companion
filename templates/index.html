The Pokémon Companion App is a full-stack project built as part of Harvard’s CS50 course. It provides tools to explore Pokémon data, build teams, and gain strategic insights for battles.

Key Features

  • Pokedex — browse all Pokémon by generation with clean sprite displays.
  • Detailed Pokémon Pages — see types, abilities, complete move tables, and evolution chains.
  • Team Builder — logged-in users can create and save multiple teams of up to six Pokémon.
  • Battle Helper — select an opponent Pokémon, and the app determines which Pokémon from each team has the best matchup.
